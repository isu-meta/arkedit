# ARKEdit

## About

ARKEdit is a GUI application for searching and editing ARKs. It supports
editing arbitrary fields and is not limited those available as part of
[EZID's metadata profiles](https://ezid.cdlib.org/doc/apidoc.html#metadata-profiles).
